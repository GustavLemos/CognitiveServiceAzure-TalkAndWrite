2-Objetivo:
	
	A função do sitema é explorar o serviço cognitivo vinculado no ambiente da Azure e
	uma aplicação simples de console de um jogo da velha.

3-Funcionamento:
	
	Basicamente o sistema funciona da seguinte forma: primeirp o sistema perguntará o
	seu nome e explicará como se joga o jogo composto por dois jogadores, cada jogado 	
	dirá um número que é a referência de cada posição existente no tabuleiro(3X3), após 
	dito o número na vez do jogador o sistema preencherá as posições do tabuleiro com
	a seguinte relação:

		1- Player 1 - representado pelo símbolo "X";
		2- Player 2 - representado pelo símbolo "O";

	Os resultados possíveis do jogo da velha são:

		1- A vitória de um dos jogadores completando uma fileira sequencial do mesmo símbolo primeiro em:
			1.1- Vertical;
			1.2- Horizontal;
			1.3- Diagonal.
		2- Empate quando a possibilidade de ambos os jogadores de compor as fileiras acima for nula.
